# Personal Portfolio Website

A fully responsive portfolio website built with HTML & CSS

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Git & GitHub

## Features

- **Responsive Design** - Works on mobile, tablet, and desktop
- **Sticky Navbar** - Navigation that sticks to the top
- **Hero Section** - Eye-catching landing with animations
- **About Page** - Skills showcase with grid layout
- **Projects Gallery** - Project cards and image gallery
- **Contact Form** - Form with HTML5 validation
- **Smooth Animations** - Hover effects and transitions

## CSS Concepts Used

✅ Box Model  
✅ Flexbox (navbar, cards)  
✅ Grid (skills, gallery) 
✅ Position (sticky, absolute)   
✅ Media Queries (mobile, tablet, desktop)  
✅ Animations & Transitions  
✅ Pseudo-classes (:hover, :focus, :nth-child)  
✅ CSS Variables  
✅ Responsive Units (%, rem, vh, vw)

## Pages

1. `index.html` - Home page with hero section
2. `about.html` - About section with skills
3. `projects.html` - Projects showcase and gallery
4. `contact.html` - Contact form

## What I Learned

- How to create responsive layouts using Flexbox and Grid
- Implementing sticky positioning for navigation
- Using CSS animations and transitions
- Form validation with HTML5
- Git workflow with branches and pull requests
- Media queries for different screen sizes
